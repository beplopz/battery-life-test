# Battery Life Test Website

A static website for Topic 15: Battery Life Test. It includes a flowchart, schematic diagram, and comparison table for measuring laptop battery life under different usage scenarios and comparing it with desktop power consumption.

## Open on a computer

Open `index.html` in a browser or serve the folder with any static web server.

## Open on a cellphone

The site is responsive and includes cellphone-friendly navigation, stacked flowchart/schematic sections, and a card-style table on small screens.

To view it from your phone while your computer and phone are on the same Wi-Fi network:

1. Run `python3 -m http.server 8000` in this folder.
2. Find your computer's local IP address, such as `192.168.1.25`.
3. On your phone, open `http://YOUR-IP-ADDRESS:8000/index.html` in the browser.

You can also upload the folder to a static host such as GitHub Pages, Netlify, or Vercel and open the hosted link on your cellphone.
